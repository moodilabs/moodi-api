# 이슈 생성 스킬

## 설명

작업 내용을 분석하여 GitHub 이슈를 생성한다.

## 참고 파일

- `CLAUDE.md` — 프로젝트 컨벤션 및 아키텍처
- `.github/ISSUE_TEMPLATE/issue.md` — 이슈 템플릿

## 절차

1. 사용자의 요청을 분석하여 이슈 제목과 본문 작성
2. 아래 템플릿 양식에 맞춰 본문 작성
3. `gh issue create`로 이슈 생성

## 이슈 템플릿

```markdown
## 설명


## 작업 범위
- [ ] 

## 참고 사항

```

## 규칙

- 이슈 제목은 간결하게 작성 (50자 이내)
- 본문은 반드시 위 템플릿 양식을 그대로 사용한다
