== Overview

=== 인증

모든 인증이 필요한 API는 요청 헤더에 Bearer 토큰을 포함해야 합니다.

[source,http]
----
Authorization: Bearer {accessToken}
----

토큰이 없거나 유효하지 않은 경우 401 응답을 반환합니다.
토큰이 만료된 경우 `/api/v1/auth/reissue` 를 통해 재발급 받아야 합니다.
